
# java-consoleGame

=========================== 지우의 모험 ===========================
1. 시작하기
    - 스토리 보여주기
    - 시작 포켓몬 고르기
    - 게임 진행 방법 설명

2. 파일 불러오기

3. 내정보 보기
    - 스텟(플레이어 레벨, 플레이어 HP)
    - 가지고 있는 게임 머니
    - 인벤토리(가지고 있는 몬스터볼 종류 및 개수)
    - 포획한 포켓몬

4. 상점(몬스터볼)
    - 몬스터 볼 (1000원)
    - 수퍼 볼  (2000원)
    - 하이퍼 볼 (3000원)
    - 마스터 볼 (4000원)

5. 포켓몬 포획하기
    - 포획하는 방법 설명
    - 포획하기(사용할 볼 선택)
    - 전투
    - 
    - 결과
6. 다음 상대(로켓단)
    - 상대할 로켓단 정보 보여주기
    - 전투
    - 결과(보상획득)

7. 휴식하기
    - 체력 회복

8. 파일저장
